Hi, I'm Lube Bot! I store quotes made by the majestic Big Lube and repeat them at your demand in Discord. I have the following commands that you can use:

!ping - Pong!
!help - Show these instructions
!add <quote> - Add a new Lube quote
!edit <id> <quote> - Edit the Lube quote at the specified id by replacing it with the new one
!remove <id> - Remove the Lube quote at the specified id
!getfulljson - Get the full json file that stores all the quotes
!get <id> - Get an existing Lube quote at the specified id
!random - Show a random Lube quote
!list - List all existing Lube quotes and their corresponding ids
